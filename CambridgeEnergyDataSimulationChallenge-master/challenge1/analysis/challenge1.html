2.Data analysis.¶

When using VAR model, it is very important to check whether a data is stationary or not. Only stationary data can be used in VAR model.

I will use Hodrick–Prescott filter to check whether the data have trend components or not. By using Hodrick–Prescott filter, we can separate(remove) a trend from a data.

note: With regards to Hodrick-Prescott smoothing parameter, I will use 129600 as it is suggested to use 129600 for monthly data. (A value of 1600 is suggested for quarterly data. Ravn and Uhlig suggest using a value of 6.25 for annual data and 129600 for monthly data.) https://en.wikipedia.org/wiki/Hodrick%E2%80%93Prescott_filter http://statsmodels.sourceforge.net/devel/generated/statsmodels.tsa.filters.hpfilter.html

As shown in graphs above, EnergyProduction, Daylight and Temperature obviously have trend components and are unstationary. (smoothed data is a data in which trend components are removed.)

As the above graphs also prove that each house has a similar wave shape, I will use smoothed data of "house 1" to take an unit root test (Augumented Dickey-Fuller test).

The unit root test tests whether a time series variable is stationary or non-stationary using an autoregressive model. As a result of the unit root test, if a P-values of variables are small enough to reject, we can reject a null hypothesis that the precess is a unit root process and can adopt alternative hypothesis that the process is stationary.

In the unit root test, I will choose ‘AIC’ as a parameter of autolag, which will decide the number of lags that minimize the information criterium.

def VAR_model(data_type, pred_file, mape_file):
    file_pred = open(pred_file, 'w')
    file_mape = open(mape_file, 'w')
    pred_writer = csv.writer(file_pred)
    pred_writer.writerow(['House','EnergyProduction'])
    sum0 = 0
    for i in data_type:
        data = data_train[data_train.House==i][['EnergyProduction','Daylight','Temperature']].set_index(rang)
        data_diff = data.diff().dropna()
        model = var.VAR(data_diff)

        sum0 += abs(data_pred-data_test.EnergyProduction[i-1])/data_test.EnergyProduction[i-1]     
        pred_writer.writerow([i,data_pred])
    mape = round(100*sum0/len(data_type),3)

    file_pred.close()
    file_mape.close()

    return mape
